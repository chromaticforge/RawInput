## Raw Mouse Input
Prevents OS-Level Processing. Powered by Oneconfig.

### Features
- Asynchronous polling. 
- Automatic rescanning.
- Safely disables on unsupported systems.

### Usage
`/rawinput` to open oneconfig gui.

`/rawinput rescan` to rescan for new mice.
